# West Quote – Monorepo

> [!NOTE]
> Special thanks to [kanye.rest](https://github.com/ajzbc/kanye.rest) for creating an awesome free api.

## Project Structure

```
./ (root)
├─ apps/
│  ├─ web/		react web app (accessible on all devices)
├─ packages/
│  ├─ load-quote/	the thing that fetches quote from kanye.rest
├─ .gitignore
├─ package.json
├─ README.md
```
